Write step5_tco.py. It is step4_if_fn_do.py with ONE change: tail call optimization in eval_.

The change: replace recursion with a while loop. At the TOP of eval_, add `while True:`. Then instead of returning from recursive calls to eval_, UPDATE ast and env and `continue`. Only `return` for non-recursive cases.

Specifically, the List branch becomes:

```python
if isinstance(ast, List):
    if not ast:
        return ast
    first = ast[0]
    if isinstance(first, Symbol):
        if first == 'def!':
            val = eval_(ast[2], env)
            env[ast[1]] = val
            return val
        if first == 'let*':
            let_env = Env(mappings={}, outer=env)
            items = list(ast[1])
            for i in range(0, len(items), 2):
                let_env[items[i]] = eval_(items[i + 1], let_env)
            ast = ast[2]
            env = let_env
            continue
        if first == 'do':
            for x in ast[1:-1]:
                eval_(x, env)
            ast = ast[-1]
            continue
        if first == 'if':
            cond = eval_(ast[1], env)
            if cond is not Nil.NIL and cond is not Boolean.FALSE:
                ast = ast[2]
            elif len(ast) > 3:
                ast = ast[3]
            else:
                return Nil.NIL
            continue
    proc = eval_(first, env)
    if isinstance(proc, Fn):
        args = [eval_(x, env) for x in ast[1:]]
        ast, env = proc.call_tco(args)
        continue
    raise Error(f'cannot apply {proc}')
```

The key insight: let*, do, and if in tail position UPDATE ast/env and continue. fn* returns a closure. When calling a fn* closure, instead of recursing, we get back new (ast, env) and loop.

To make this work, fn* must return a closure that can either:
- Return a final result directly
- Return a tuple (new_ast, new_env) for TCO

Use this pattern for fn*:
```python
if first == 'fn*':
    params, body = ast[1], ast[2]
    def make_closure(captured_env):
        def closure(args):
            return (body, call_env(captured_env, params, args))
        closure.is_mal_fn = True
        return closure
    return Fn(make_closure(env))
```

Actually that's too complex. Use a simpler approach: wrap the TCO result in a sentinel class:

class TCOResult:

Wait, that's NOT TCO. The fn* closure still calls eval_ recursively. For TCO, the fn* call should NOT recurse — it should return the new (ast, env) to the while loop.

Here is the CORRECT TCO approach. The fn* closure returns a special marker instead of calling eval_:

```python
class _TCO:
    __slots__ = ('ast', 'env')
    def __init__(self, ast, env):
        self.ast = ast
        self.env = env
```

Then in the function call:
```python
proc = eval_(first, env)
if isinstance(proc, Fn):
    args = [eval_(x, env) for x in ast[1:]]
    result = proc.call(args)
    if isinstance(result, _TCO):
        ast = result.ast
        env = result.env
        continue
    return result
```

And fn*:
```python
if first == 'fn*':
    params, body = ast[1], ast[2]
    def make_closure(captured_env):
        def closure(args):
            return _TCO(body, call_env(captured_env, params, args))
        return closure
    return Fn(make_closure(env))
```
